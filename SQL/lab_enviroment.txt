# DVWA Setup Guide

## Requirements
- Apache / Nginx
- PHP
- MySQL / MariaDB
- Kali Linux (Recommended)

## Installation Steps
```bash
sudo apt update
sudo apt install dvwa


----------------------------------------------------------


Configuration

Edit /etc/dvwa/config.inc.php

Set database credentials

Start Apache & MySQL

sudo systemctl start apache2
sudo systemctl start mysql

--------------------------------------------------------------



DVWA Security Level

Set security level to Low for SQL Injection testing.

Access DVWA
http://localhost/dvwa
