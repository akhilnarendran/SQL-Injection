# SQL Injection Methodology

## Steps Followed
1. Identify input fields
2. Test for SQL errors
3. Confirm injectable parameters
4. Automate exploitation using SQLMap
5. Enumerate database
6. Extract sensitive data
7. Analyze impact
8. Suggest mitigations

## Injection Types
- Error-based
- Union-based
- Boolean-based
- Time-based

## Tools
- Manual payloads
- SQLMap automation
